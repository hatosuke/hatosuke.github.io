プライバシーポリシー

Sugar Scale（シュガースケール）（以下「当アプリ」）は、ユーザーのプライバシーを尊重し、個人情報の適切な取り扱いに努めます。本ポリシーでは、当アプリにおける情報の収集・利用・共有について説明します。

1. 収集する情報

当アプリは、以下の情報・権限を利用します。

  • カメラ（NSCameraUsageDescription）：食品を撮影して糖質計算に利用するため
  • フォトライブラリ（NSPhotoLibraryUsageDescription）：既存の食品写真を選択するため
  • AI解析機能（オプション）：ユーザーが明示的に同意した場合のみ、食品の写真をAI解析のため Firebase Cloud Functions へ送信します
  • 匿名認証：Firebase Anonymous Authentication を使用し、個人を特定しない UID で利用状況を管理します

当アプリは、氏名・メールアドレス・電話番号など個人を特定する情報を一切収集しません。

AI解析用に送信された画像の保存期間・削除タイミング・第三者AIプロバイダへの再送有無は、現時点で未確認です。詳細はお問い合わせください。

2. 外部サービスの利用

当アプリは品質改善と機能提供のために、以下のサービスを利用します。

  • Firebase Crashlytics：匿名化されたクラッシュ情報を収集
  • Firebase Analytics：匿名化された利用統計を収集
  • Firebase Authentication：匿名認証（個人を特定しない）
  • Firebase Firestore：AI解析利用回数の管理（匿名UID紐付け）
  • Firebase Cloud Functions：AI解析処理（ユーザー同意時のみ画像を送信）
  • Google AdMob：広告表示（AdMob のポリシーに準拠）

公式情報: Firebase のプライバシーとセキュリティ / Google 広告ポリシー（AdMob関連）

3. データ保存について

食事記録・食品マスタなどの主要データは、デバイス内ローカルストレージに保存されます。

  • AI解析機能を使用する場合のみ、ユーザーが同意した写真が Cloud Functions へ送信されます
  • AI解析以外の食事記録データは端末内で管理されます

4. 第三者提供

当アプリはユーザーの個人情報を第三者に提供しません。ただし、Crashlytics・Analytics・AdMob・Cloud Functions 等の利用に伴い、匿名化された技術データが各サービス提供元に送信される場合があります。

5. データ削除

アプリをアンインストールすると、端末内に保存された当アプリのデータは削除されます。
